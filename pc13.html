PC 13

Práctica crítica 13

PRE POP test de disparo de válbula PSV

La práctica crítica PC-13 garantiza que las válvulas de seguridad funcionen correctamente mediante pruebas de disparo y análisis de presión. Se aplica a válvulas en tanques, autotanques y sistemas de GLP. El procedimiento incluye inspección visual, desmontaje, transporte, prueba Pre-Pop, mantenimiento y verificación de calibración. Si una válvula falla, se investiga el incidente y se toman acciones correctivas. Todo el proceso debe ser realizado por personal certificado, siguiendo normas API y regulaciones locales para asegurar la protección de los sistemas presurizados.
